XDownloader Help

General

XDownloader moves or copies files from one place to another, while renaming them according to a template. This template contains variables, which is the different pieces of EXIF information found in the file. Common uses are renaming files to the date/time captured, and perhaps the image number to guarantee uniqueness.

Check out the possible template variables that are available.

Using Job Names

You can include the special variable {JobName} in the renaming template, and check the "Ask for job name" box. Whenever you click Download, you will be asked for a job name. The job name can be any string you find descriptive, like subject matter, client name, geographic location, etc.
